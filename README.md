# Task 3 - Cats vs Dogs Classification

## Objective
Classify images of cats and dogs using 
Support Vector Machine (SVM).

## Dataset
Dogs vs Cats (Kaggle)
- 25,000 labeled images
- 2,000 images used for training

## Model
- Algorithm: SVM (RBF Kernel)
- Images: 64x64 Grayscale
- Classes: Cat, Dog

## Libraries Used
- Python, NumPy, OpenCV
- Scikit-learn, Matplotlib# PRODIGY_ML_03
